# 居住地・可住地人口密度作成手順書

# データ
土地利用情報：
国土数値情報（2021年）
土地利用細分メッシュデータ（100m）：https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-L03-b-2021.html

人口データ：
国土数値情報（2020年国勢調査を用いて500mメッシュデータにしたもの。2024年作成版）
500mメッシュ別将来推計人口データ（R6国政局推計）：https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-mesh500r6.html

# 作成手順
土地利用細分メッシュデータ（全国）をダウンロード
↓
都道府県ごとに、郵便番号ポリゴンと土地利用細分メッシュデータ（100m）を重ねて、各郵便番号における土地利用データ（居住地面積・可住地面積）を作成
↓
郵便番号ポリゴンと人口データ（500mメッシュ）を重ねて、各郵便番号における人口数データ（面積按分）を作成
↓
各郵便番号の土地利用データ（居住地面積・可住地面積）と人口数データを結合して、各郵便番号の居住地人口密度、可住地人口密度を算出
